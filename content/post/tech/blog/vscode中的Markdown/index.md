## Markdown All in One插件

## 比较实用的功能


- <kbd>Ctrl</kbd> + <kbd>B</kbd>  -> **文字加粗**

- 打开命令行，输入`toc`, 根据多级标题实现目录功能

- 先选中文本，再粘贴链接，快速创建超链接

- <kbd>Alt</kbd> + <kbd>Shift</kbd> + <kbd>F</kbd> -> 表格文本格式化（惊奇）
